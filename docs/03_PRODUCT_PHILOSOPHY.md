# Companion Philosophy

Military Companion is not a tool.

It is a companion.

The difference is intentional.

A tool waits for instructions.

A companion understands context, offers guidance, and stays with the soldier throughout their journey.

Every feature should feel like part of an ongoing relationship rather than an isolated utility.

Military Companion should never feel like a government website or a collection of calculators.

Instead, it should feel like an experienced senior who genuinely wants every soldier to succeed.

---

# Progressive Personalization

Military Companion should begin with broad guidance.

As soldiers continue using the application, it should naturally learn:

- Goals
- Interests
- Career direction
- Questions
- Saved opportunities
- Reading habits
- Completed milestones

Personalization should happen gradually.

The application should never overwhelm users with lengthy onboarding forms.

Instead, it should quietly become more helpful over time.

Our goal is to earn understanding rather than demand information.
